# XIRR 年化報酬率 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Calculate per-stock and portfolio-level XIRR (Extended Internal Rate of Return / 年化報酬率) from existing transaction and dividend data, expose via the existing `/api/portfolio/summary` endpoint, and display in the Angular dashboard.

**Architecture:** XIRR cash flows are derived by replaying buy (negative outflow) / sell (positive inflow) / dividend (positive inflow) history per symbol, with a terminal value equal to current market value at today's date. The `pyxirr` library handles the numerical solver. The result is added as an optional field (`xirr`) on `StockHolding` and `PortfolioSummary` so the existing API contract is backward-compatible.

**Tech Stack:** Python 3.13, FastAPI, pyxirr, Angular 21, PrimeNG

Add this function near the top of `portfolio_service.py`, after the existing imports and before `sanitize_symbol`:

```python
from datetime import date as date_type
from typing import List, Optional, Tuple

def _calculate_xirr(cash_flows: List[Tuple[date_type, Decimal]]) -> Optional[Decimal]:
    """
    Compute XIRR from a list of (date, amount) pairs.
    Returns None if calculation is impossible or fails.
    - amounts < 0: outflows (buy)
    - amounts > 0: inflows (sell, dividend, terminal market value)
    """
    if len(cash_flows) < 2:
        return None
    
    # All same date → undefined
    dates = [cf[0] for cf in cash_flows]
    if len(set(dates)) < 2:
        return None

    # Terminal value is the last flow; if it's 0 we can't compute
    if cash_flows[-1][1] <= Decimal("0"):
        return None

    try:
        from pyxirr import xirr as _xirr
        result = _xirr(
            [cf[0] for cf in cash_flows],
            [float(cf[1]) for cf in cash_flows],
        )
        if result is None or not isinstance(result, float):
            return None
        import math
        if math.isnan(result) or math.isinf(result):
            return None
        return Decimal(str(round(result, 6)))
    except Exception:
        return None
```

### Task 5: Wire XIRR into get_portfolio_summary

- [ ] **Step 1: Understand the existing cash flow data available**

In `get_portfolio_summary`, transactions are already iterated in date order. We need to build cash flow lists per symbol:
- BUY → negative: `-(quantity * price + fee)`  
- SELL → positive: `+(quantity * price - fee - tax)`
- Dividend → positive: `+amount`
- Terminal: `+(market_value)` at today's date
